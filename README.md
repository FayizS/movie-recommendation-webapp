# Movie Recommendation System

A full-stack movie web app that displays trending movies from TMDb and shows personalized recommendations using machine learning.

## Features
- Live movie data from TMDb API
- User click tracking
- Logistic Regression model trained on user interactions
- Personalized movie ranking
- Flask backend + HTML/CSS frontend

## Tech Stack
- Python, Flask
- SQLite
- Pandas, Scikit-learn
- TMDb API
- HTML, CSS

## How it works
1. Movies are loaded from TMDb
2. User clicks are stored in SQLite
3. A machine learning model learns what users prefer
4. Movies are ranked based on predicted interest
